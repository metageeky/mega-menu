Accessible Mega Menu

Design Specifications

Design Specifications Link

The Accessible Mega Menu was designed to meet the specific needs of a website redesign. A mega menu approach was desired as the site is a library and thus offers a variety of services that need to be discoverable by users without having to do deep diving. The current menu structure reflects a draft of the proposed site "structure."

The mega menu design request specified the following features:

  • The overall menu conforms and possibly exceeds accessibility standards.
  • The mega menu includes a mobile-friendly mode that is not necesarily the same as the "desktop" mode.
  • In desktop mode, the menus open on hover to increase discoverability.
  • The individual mega menus consist of up to 4 columns that are either an image link or a column of text links. The text link columns may have textual headings for easier cognition. Text links may span more than one column.

Interaction Logic

Interaction Logic Link

The biggest challenge with the design specifications involved the desktop view and properly balancing the interactions involving opening the menus via hover or direct manipulation. To accomplish this, the mega menu exists in one of three states:

Closed
No sub-menu in the mega menu is opened.
Hover Opened
A sub-menu is currently opened via hover interaction.
Focus Opened
A sub-menu is open and focus is currently within the mega menu.
  • Regardless of whether a menu was opened by hovering or by direct manipulation, the Escape key always closes the menu.
    • If the focus is currently within a sub-menu, the focus must be moved to the sub-menu's toggle button.
  • Focus should always override hover. If the current focus or document.activeElement is a element of the mega menu, menus will not auto-open or auto-close based on hovering over the menu.
  • No auto-open on focus. Open must be triggered by
  • If focus is IN a submenu, no hover opening unless there is a click
  • if focus is ON submenu trigger and submenu opened, hover does not open. Click outside will close.
  • If focusable, go there but if hover is open but focus moves, close the hover
